# Graduation-Thesis-Project

This repo maintains my and my partner's graduation thesis project. We have passed with 8.5/10 a.k.a 4.0/4.0, about "Text Detection in Images Using Machine Learning Methods"

Description details have been written in the final-thesis-article.pdf

-> This repo using Colab Notebook form Google, so there were some details will wrong!

-------------------------------------------------------------------------------------------------------------------
How to run this code:
1. Clone repo:
```bash
pip clone https://github.com/lggian/anluong.graduation-thesis-project
```
2. Run source1.py with IIIT5K_coco.rar
```bash
python source1.py
```
3. Run source2.py and Chars74K
```bash
python source2.py
```
Please be careful with this line in source2.py!
```bash
url = 'http://www.ee.surrey.ac.uk/CVSSP/demos/chars74k/EnglishImg.tgz'
output_path = 'EnglishImg.tgz'
```
